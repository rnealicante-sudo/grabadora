# Cartuchera Móvil 📻🎚️

**Cartuchera Móvil** es un sistema profesional de emisión de radio optimizado para dispositivos móviles y entornos de producción ágiles. Inspirado en sistemas de hardware de radio de alta gama, ofrece una interfaz táctil ergonómica para el disparo instantáneo de audios, efectos y cortinas.

## 🚀 Características Principales

- **8 Cartucheras Ergonómicas:** Diseñadas para caber en una sola pantalla sin necesidad de desplazamiento.
- **Interfaz Profesional:** Estética inspirada en consolas de radio reales con colores azul marino y gris industrial.
- **Control de Reproducción Instantáneo:** Dispara audios directamente haciendo clic en el número de la cartuchera.
- **Barra de Desplazamiento Integrada:** Ajusta la posición del audio (seek) directamente desde el cuerpo de cada cartucho.
- **Temporizadores de Alta Visibilidad:** Relojes digitales en azul brillante con modos de tiempo transcurrido y restante.
- **Master Fader:** Control de volumen general accesible desde la parte inferior.
- **Selección de Salida:** Configura el dispositivo de salida de audio directamente desde la aplicación.
- **Funciones Avanzadas:** Soporte para bucles (Loop) y encadenamiento de cartuchos (Link).

## 🛠️ Stack Tecnológico

- **Framework:** React 19 + Vite
- **Motor de Audio:** Web Audio API (Baja latencia)
- **Iconos:** Lucide React
- **Animaciones:** Framer Motion
- **Estilos:** Tailwind CSS v4

## 📱 Optimización Móvil

La aplicación ha sido diseñada específicamente para:
- Pantallas táctiles de cualquier tamaño.
- Operación con una sola mano (Master Fader inferior).
- Carga rápida y bajo consumo de recursos.

## 📖 Instrucciones de Uso

1. **Cargar Audio:** Haz clic en el botón central "CARGAR AUDIO" de cualquier cartuchera vacía.
2. **Reproducir:** Haz clic en el número grande de la izquierda o en el botón de reproducción.
3. **Ajustar Posición:** Desliza la barra central para moverte por el audio.
4. **Configuración:** Usa los iconos de bucle (Loop) o rayo (Link) para automatizar la reproducción.
5. **Pánico:** Usa el botón de parada general en la cabecera para detener todos los audios instantáneamente.
